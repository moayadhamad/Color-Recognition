# Color Recognition using OpenCV

## Description
This project uses OpenCV to detect colors in an image. It identifies predefined colors, draws a bounding box around them, and displays the color name.

## Tools
- Python
- OpenCV
- NumPy

## Files
- `color_recognition.py`
- `pngtree-6-color-bubble-dialogs-picture-image_4086768.png`
## How to Run

1. Install the required libraries:
```bash
pip install opencv-python numpy
```

2. Run the project:
```bash
python color_recognition.py
```
